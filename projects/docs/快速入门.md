# PrisonSIS 快速入门

> 从打开程序到完成一份正式笔录，只需 6 步。

---

## 准备工作

### 首次登录

1. 打开程序，进入登录界面
2. 输入分配给你的用户名和密码（由管理员提供）
3. **如果是首次登录**，程序会提示你设置 PIN 码（6位数字）：
   - 这个 PIN 用来保护你的数字签名
   - 请务必记住，忘记将无法恢复
   - 不要写在电脑上，用纸笔记录下来放在安全的地方
4. 登录成功后，进入主界面

---

## 第 1 步：认识主界面

登录后，主界面分为两个区域：

```
┌─────────────────────────────────────────────┐
│  顶部导航栏（点击切换功能）                    │
├──────────┬──────────────────────────────────┤
│          │                                   │
│ 左侧导航 │        主工作区                    │
│          │                                   │
└──────────┴──────────────────────────────────┘
```

**左侧导航包括：**

| 菜单 | 你可以在这里做 |
|------|--------------|
| 核心业务 → 笔录制作 | 新建、编辑、提交笔录（主要工作区） |
| 核心业务 → 罪犯信息 | 查看和管理罪犯档案 |
| 核心业务 → 审批中心 | 审核同事提交的笔录 |
| 核心业务 → 档案管理 | 查看已归档的正式笔录 |
| 统计分析 → 用户管理 | 管理用户账号 |
| 资源配置 → 数据备份 | 备份和恢复数据 |

---

## 第 2 步：新建一份笔录

### 2.1 打开笔录编辑器

点击左侧菜单：**核心业务 → 笔录制作**

### 2.2 选择笔录类型

在"笔录基本信息"区域，找到"笔录类型"下拉框，选择你要做的笔录类型：

```
RT-01 入监笔录    ← 新罪犯入监时做
RT-02 日常谈话    ← 平时找罪犯谈话
RT-03 案件调查    ← 调查具体案件
RT-04 行政处罚告知
RT-05 减刑假释评估
RT-06 出监教育
RT-07 医疗检查
RT-08 亲属会见
```

选择类型后，程序会自动填入该类型对应的模板内容。

### 2.3 关联罪犯

点击"选择罪犯"按钮，从罪犯列表中选择本次笔录对应的罪犯。

> 如果罪犯不在列表里，需要先在"罪犯信息管理"中添加该罪犯档案。

### 2.4 填写审讯信息

- **审讯时间**：默认填当前时间，可以修改
- **审讯地点**：从下拉框选择，或填"其他"后手动输入
- **在场人员**：输入在场人员姓名，多人用顿号（、）分隔

### 2.5 编辑笔录正文

在下方的大文本框里，根据模板内容填写审讯信息。

**编辑技巧：**

| 操作 | 方法 |
|------|------|
| 加粗文字 | 选中文字，按 `Ctrl+B`，或点击工具栏的 **B** |
| 倾斜文字 | 选中文字，按 `Ctrl+I`，或点击工具栏的 **I** |
| 保存草稿 | 按 `Ctrl+S`，或点击右下角"保存草稿"按钮 |
| 打印笔录 | 点击"打印笔录"按钮（后面会用到） |

**关于信签纸主题：**
工具栏上方有一排彩色圆点，点击可以切换编辑区的背景样式：
- 🔵 蓝色文件纸（默认，推荐日常使用）
- ⚪ 白色信签纸（经典）
- 🟡 米黄色档案纸（传统感）
- 🟢 绿色护眼纸（适合长时间编辑）
- ⬛ 深色夜间模式（夜间值班时用）

---

## 第 3 步：保存草稿

编辑过程中，程序每 5 分钟自动保存一次草稿。

**手动保存：**
- 按 `Ctrl+S`
- 或点击右下角"保存草稿"按钮

**保存成功后**，左下角会显示"上次保存：XXX"，状态变为"草稿（已保存）"。

> **草稿阶段可以随时修改，格式、内容都随意调整。**

---

## 第 4 步：打印笔录

当笔录内容填写完毕、内容确认无误后，点击右下角的**"打印笔录"**按钮。

程序会生成一份排版整齐的 PDF 文件。

**打印流程：**
1. 点击"打印笔录" → 程序生成 PDF 预览
2. 选择打印机（通常是连接了打印机的电脑）
3. 确认打印份数
4. 点击打印

**拿到纸质版后：**
1. 民警在"讯问人签字"处签字
2. 记录员在"记录员签字"处签字
3. 被讯问人（罪犯）在"被讯问人签字"处签字并按手印
4. 每页都需要签字按手印

> ⚠️ **重要：纸质版上的亲笔签名和手印，是最有法律效力的部分。一定要完整签字按手印，缺一不可。**

---

## 第 5 步：上传扫描件（最关键的一步）

纸质版签字按手印完成后，用扫描仪把纸质版扫描成 PDF 文件。

### 5.1 扫描建议

- 分辨率：建议 300 DPI，足够清晰即可
- 格式：PDF
- 扫描模式：彩色（保留签字颜色信息）
- 确保每一页都扫描完整，不要漏页

### 5.2 上传扫描件

在笔录编辑器里，点击**"上传扫描件"**按钮：

1. 在弹出的文件选择框里，选择刚才扫描得到的 PDF 文件
2. 程序会提示你输入 **PIN 码**（第1步设置的6位数字）
3. 点击"确认上传"

**程序会自动完成以下操作（你不需要做任何额外操作）：**
```
正在处理...
 ✓ 计算数字指纹（确保文件以后能发现篡改）
 ✓ 对扫描件进行加密存储（只有你能看）
 ✓ 加上你的数字签名（证明是你上传的）
 ✓ 记录签名时间和笔录编号
 ✓ 通知网络内其他节点：本笔录已正式签名
完成！
```

### 5.3 上传成功的标志

上传完成后，笔录状态会从"草稿"变为**"已签名扫描件"**（显示🔒图标）。

这意味着这份笔录正式进入"不可更改"状态。

> 🔐 **从现在起，这份笔录的数字指纹已被锁定。**
> 任何人（包括程序员、管理员）都无法在不动你的电脑和 PIN 码的前提下修改这份笔录。
> 将来如果有人质疑这份笔录，可以随时在系统里验证：指纹对得上，就是真的；指纹对不上，就是被改过的。
> 详细内容见《安全性说明》。

---

## 第 6 步：提交审批

扫描件上传完成后，点击右下角的**"提交审批"**按钮。

程序会将笔录提交给一级审批人（管教组长）审核。

**审批流程：**
```
民警提交 → 一级审批（管教组长）→ 二级审批（如需）→ 归档
```

审批状态可以在**"审批中心"**查看：
- 🟡 待审批 — 等待审核中
- 🟢 通过 — 审批通过，正式归档
- 🔴 驳回 — 被驳回，需要修改后重新提交

## 进阶功能

### 查看正式笔录

点击**"核心业务 → 档案管理"**，可以查看所有已归档的正式笔录。

已归档的笔录只能查看，无法修改。每一份归档笔录都显示了：
- 笔录编号
- 关联罪犯
- 签名时间
- 数字指纹（可验证文件是否被篡改）

### 恢复误删的内容

如果误删了一份草稿，可以这样做：
1. 点击"核心业务 → 笔录制作"
2. 查看"草稿箱"区域（右下角）
3. 草稿箱里列出了你最近保存的所有草稿
4. 双击任意一行，即可恢复加载该草稿

### 从其他工作站下载扫描件

如果在其他工作站上传了某份笔录的扫描件，可以在当前工作站下载：

1. 在笔录编辑器中，点击**"📥 下载扫描件"**按钮
2. 弹出下载对话框，显示 P2P 网络中各节点持有的扫描件索引
3. 选择要下载的扫描件行，输入你的 PIN 码
4. 程序会从持有该扫描件的节点拉取加密密文，并自动解密
5. 点击"导出 PDF"将解密后的文件保存到本地

> **注意**：下载功能需要 P2P 网络正常（状态栏显示"在线"），且对方节点在线才能拉取。

## 常见问题

**问：电脑突然关机了，笔录还在吗？**
答：还在。程序每5分钟自动保存一次草稿，重新打开程序后在"草稿箱"里可以找到。

**问：已经上传扫描件了，发现内容有错字，能改吗？**
答：已签名的扫描件不能修改。请在"档案管理"中新建一份笔录重新制作。**这就是为什么建议先仔细检查内容，再上传扫描件。**

**问：PIN 码忘了怎么办？**
答：请联系管理员重置。注意：重置后之前的数字签名将无法使用，需要重新签名。已上传的正式扫描件不受影响（签名本身已存在，无法撤销）。

**问：内网断了，还能用吗？**
答：完全可以用。笔录编辑、保存、扫描件上传都在本地完成，连接恢复后 P2P 网络自动恢复。
